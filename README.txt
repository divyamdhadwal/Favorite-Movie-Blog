Movie Website
This is a simple movie trailer website which consists information and trailer of six of my favourite movies. Upon clicking the movie title or movie poster, one can watch the youtube trailer of the respective movie.

Documentation
Use  this command to see the documentation of the class used in the project.

print movies.__doc__

Project contents
Media.py
entertainment_center.py
fresh_tomatoes.py

Install

Prerequisite 
Python 2.7.13 must be preinstalled on the system. 

Operating Instructions
Make a folder and put all the files into it (media.py, entertainment_center.py, fresh_tomatoes.py)
Execute entertainment_ centre.py using python shell or directly open it up using IDLE and click “run”.
Web browser will automatically open with the page containing the movie trailers.
A HTML file will be created automatically , as a result of execution.
